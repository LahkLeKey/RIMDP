# RIMDP

Repair Intelligence & Maintenance Decision Platform monorepo.

## Test & Coverage Scaffold

This repository is configured with Vitest projects for:

- `packages/shared`
- `apps/api`
- `apps/web`

### Run tests

```bash
pnpm test
```

### Run tests with coverage

```bash
pnpm test:coverage
```

### 100% coverage gates

Coverage thresholds are set to **100%** (`lines`, `branches`, `functions`, `statements`) on scoped targets per package via:

- `packages/shared/vitest.config.ts`
- `apps/api/vitest.config.ts`
- `apps/web/vitest.config.ts`

Root test orchestration is configured in:

- `vitest.config.ts`

## Notes

- Coverage HTML reports are generated by Vitest in each package test run.
- Expand each package `coverage.include` list as you add tests to progressively enforce 100% across more modules.

## Docker Web Service

The frontend can run in Docker Compose as an optional profile:

```bash
docker compose --profile web up -d web
```

- Web URL: `http://localhost:5173`
- The container points to API at `http://host.docker.internal:4000` by default.

To stop it:

```bash
docker compose --profile web down
```
